# Hirely – Job Portal

## 🚀 Overview

Hirely is a full-stack job portal where recruiters can post jobs and candidates can search and apply.

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Spring Boot
* Database: MySQL

## ✨ Features

* User registration & login
* Job posting (Admin/Recruiter)
* Job search & apply
* REST API integration

## ⚙️ How to Run Locally

### Backend

* Go to `job-backend`
* Run the Spring Boot application

### Frontend

* Go to `job-frontend`
* Run:

```
npm install
npm start
```
